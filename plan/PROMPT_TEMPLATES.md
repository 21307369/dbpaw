# AI 协作 Prompt 模板库

> 复制以下模板，填入任务信息，直接发送给 AI Agent

---

## 🚀 启动新任务

```markdown
## 开始任务

我正在进行 Chat2DB 的 Rust/Tauri 重写项目。现在要开始执行以下任务：

**任务 ID**: [填写，如 TASK-1-1]
**任务名称**: [填写]

### 任务 Spec

[从 TASKS.md 复制完整的任务 Spec]

---

### 请求

1. 首先确认你理解了这个任务的要求
2. 列出实现步骤
3. 逐步实现，每个关键步骤后暂停让我验证
4. 完成后帮我检查验收标准
```

---

## 📂 提供项目上下文

```markdown
- TASK-X-X: [简述]

请继续 TASK-X-X 的实现。
```

## 实用技巧

### 1. 保持上下文连贯
每次新对话开始时，简要说明：
- 项目是什么
- 当前进度
- 本次要完成什么

### 2. 分步验证
不要一次让 AI 完成太多代码，每完成一个函数/组件就验证一下。

### 3. 明确说"停"
如果 AI 生成了太多代码，说：
> "暂停，我先验证到这里，确认无误后再继续。"

### 4. 追问细节
如果 AI 的解释不够清楚：
> "这部分我不太理解，能用更简单的语言解释一下吗？"

### 5. 要求解释
在接受代码之前：
> "请解释一下这段代码每一部分的作用，特别是 [某个复杂的部分]"

### 6. 保存进度
完成每个任务后，在 TASKS.md 中更新状态，记录：
- 实际产出的文件
- 遇到的问题
- 与 Spec 的偏差（如有）
